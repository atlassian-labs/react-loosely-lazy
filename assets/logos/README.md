# React Loosely Lazy Logos

This directory contains the official logo variations for the React Loosely Lazy project.

## Logo Variations

1. `logo-default.svg` - Full color version with React blue (#61DAFB) and Atlassian blue (#0052CC)
2. `logo-mono.svg` - Monochrome version in black, suitable for print or single-color applications
3. `logo-inverted.svg` - White version for use on dark backgrounds

## Usage

- For general use, prefer the default logo
- Use the monochrome version when color printing is not available or for single-color applications
- Use the inverted version when placing the logo on dark backgrounds

The logo represents the core concepts of React Loosely Lazy:
- The outer circle represents React's component architecture
- The spiral pattern represents the "loosely" concept of our lazy loading approach
- The center dot represents the focused, performant core of the library
